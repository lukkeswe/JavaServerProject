Save File Handler Flowchart

The Save File Handler parse recived text content and writes it in correct format to the user's account.

flowchart TD A(Start handle) --> B{Request method == POST?} B --> |No| C[Send 405 Method Not Allowed] B --> |Yes| D[Get Java session ID] C --> T(End) D --> E[Get username from SessionManager] E --> F{User == null?} F --> |Yes| G[Send 403 Forbidden
Close response body] F --> |No| H[Init userPath] G --> T H --> I{User == Admin?} I --> |Yes| J[Build path to Admin's path] I --> |No| K[Build path to user's path] J --> L[Append filename to path] K --> L L --> M{Write file to path} M -->|Fail| N[Send 403 forbidden] M -->|Success| O[response = Success] N --> T O --> P[Send heasers 200 and response bytes] P --> T
