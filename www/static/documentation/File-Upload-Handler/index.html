Upload File Handler Flowchart

The Upload File Handler uploads files to user's accounts.

flowchart TD A[Start handle] --> B{Request method == POST?} B --> |No| C[Send 405 Method Not Allowed] B --> |Yes| D[Get Java session ID] C --> T[End] D --> E[Get username from SessionManager] E --> F{User == null?} F --> |Yes| G[Send 403 Forbidden
Close response body] F --> |No| H{Try block} G --> T H --> I[Log Receiving upload...] I --> J[Get request headers] J --> K[Get Content-Type from headers] K --> L[Read request body as bytes] L --> M[Extract headers from body] M --> N[Extract body content type] N --> O[Log bodyContentType] O --> P[Call handleMultipartFormData...] P --> Q[Convert response to bytes] Q --> R[Send 200 OK
Write response to OutputStream] R --> S[Close OutputStream] S --> T H -->|Exception| U[Catch Exception] U --> V[Log stack trace] V --> W[Create response: Upload failed: + e.getMessage] W --> X[Send 500 Internal Server Error] X --> Y[Write response to OutputStream] Y --> Z[Close OutputStream] Z --> T
