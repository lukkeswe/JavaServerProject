Delete Folder Handler Flowchart

The Delete Folder Handler removes a folder/directory and all it's contents from the user's account.

flowchart TD A(Start handle) --> B{Request method == POST?} B --> |No| C[Send 405 Method Not Allowed] B --> |Yes| D[Get Java session ID] C --> T(End) D --> E[Get username from SessionManager] E --> F{User == null?} F --> |Yes| G[Send 403 Forbidden
Close response body] F --> |No| H[Init userPath] G --> T H --> I{User == Admin?} I --> |Yes| J[Build path to Admin's path] I --> |No| K[Build path to user's path] J --> M{Remove targeted path/directory using deleteFolder helper method} K --> M M -->|Fail| N[Send headers 200 and response Fail] M -->|Success| O[Send headers 200 and response Success] O --> T N --> T
