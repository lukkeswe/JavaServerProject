Delete File Handler Flowchart

The Delete File Handler removes a targeted file from the user's account.

Close response body] F --> |No| H[Init userPath] G --> T H --> I{User == Admin?} I --> |Yes| J[Build path to Admin's path] I --> |No| K[Build path to user's path] J --> L[Append filename to path] K --> L L --> M{Get targeted file} M -->|Fail| N[Send 403 forbidden] M -->|Exist| O{Remove file} N --> T O -->|Success| P[Send heasers 200 and Success msg] O -->|Fail| Q[Log: error] P --> T Q --> T
