Upload Folder Handler Flowchart

The Upload Folder Handler uploads folders with web relted files to the user's accounts

flowchart TD A[Start handle] --> B{Request method is POST} B -->|No| C[Send 405 Method Not Allowed] B -->|Yes| D[Get Java session ID] D --> E[Get username from SessionManager] E --> F{User is null} F -->|Yes| G[Send 403 Forbidden] F -->|No| H{Try block} H --> I[Print Reciving upload] I --> J[Get request headers] J --> K[Get content type] K --> L[Read request body bytes] L --> M[Call handleMultipartFormDataFolder] M --> N[Prepare response bytes] N --> O[Send 200 OK] O --> P[Write response to output stream] P --> Q[Close output stream] H -->|Exception| R[Print stack trace] R --> S[Prepare error message Upload failed] S --> T[Send 500 Internal Server Error] T --> U[Write error response] U --> V[Close output stream] C --> W[End] G --> W Q --> W V --> W
