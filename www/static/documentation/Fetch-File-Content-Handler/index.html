Fetch File Content Handler Flowchart

The Fetch File Content Handler collects bytes from text files in the user's account.

Close response body] F --> |No| H[Init userPath] G --> T H --> I{User == Admin?} I --> |Yes| J[Build path to Admin's path] I --> |No| K[Build path to user's path] J --> L[Append filename to path] K --> L L --> M{Get path} M -->|Invalid path| N[Log: File not found
Send 403 forbidden] M -->|File exist| O[Collect file's bytes into response] N --> T O --> P[Send heasers 200 and response bytes] P --> T
