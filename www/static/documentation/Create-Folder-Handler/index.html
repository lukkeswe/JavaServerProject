Create Folder Handler Flowchart

The Create Folder Handler creates a new folder/directory in the user's account.

flowchart TD A(Start handle) --> B{Request method == POST?} B --> |No| C[Send 405 Method Not Allowed] B --> |Yes| D[Get Java session ID] C --> T(End) D --> E[Get username from SessionManager] E --> F{User == null?} F --> |Yes| G[Send 403 Forbidden
Close response body] F --> |No| H[Init userPath] G --> T H --> I{User == Admin?} I --> |Yes| J[Build path to Admin's path] I --> |No| K[Build path to user's path] J --> M{Make directory from targeted path} K --> M M -->|Fail| N[response = Fail] M -->|Success| O[response = Success] O --> P[Send heasers 200 and response bytes] N --> P P --> T
